üöÄ QUICK START
--------------

Local Build:
  cd doc
  pip install -r requirements.txt
  make clean html
  open _build/html/index.html

Auto-Rebuild:
  pip install sphinx-autobuild
  sphinx-autobuild . _build/html

Cloud Deployment:
  Push to GitHub ‚Üí Connect to ReadTheDocs ‚Üí Auto-deploy

üìÅ KEY FILES
-----------

conf.py
  Main Sphinx configuration
  Edit for: project name, theme options, extensions

_static/custom.css
  Custom styling (465 lines)
  Edit for: colors, fonts, layout, responsive design

_templates/layout.html
  Custom HTML layout
  Edit for: footer, header, meta tags

requirements.txt
  Python dependencies
  Update for: new extensions, version pins

.readthedocs.yml
  ReadTheDocs configuration
  Edit for: build settings, environment

üéØ CUSTOMIZATION
----------------

Change Colors:
  1. Edit conf.py (color_primary, color_secondary)
  2. Edit _static/custom.css (:root variables)

Add Fonts:
  Edit _static/custom.css (font-family rules)

Change Footer:
  Edit _templates/layout.html (footer block)

Add New Page:
  1. Create file.rst
  2. Add to index.rst toctree
